# Configuration des fichiers d'environnement

## `bdd.env`

```env
BDD_DATABASE=
BDD_USERNAME=
BDD_PASSWORD=
BDD_DRIVER=
BDD_HOSTNAME=
```

> Exemple

```env
BDD_DATABASE=beeland
BDD_USERNAME=beeland
BDD_PASSWORD=password
BDD_DRIVER=mariadb
BDD_HOSTNAME=127.0.0.1
```
